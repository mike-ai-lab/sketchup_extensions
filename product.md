# Product Overview

Markdown Live Preview (DocMark) is a web-based markdown editor with real-time preview capabilities. The application provides a split-pane interface where users can write markdown on the left and see the rendered output on the right.

## Core Features

- Live markdown preview with syntax highlighting
- Bidirectional cursor and scroll synchronization between editor and preview
- PDF export with configurable font settings
- Multiple preview styles (GitHub, GitBook, VSCode)
- Dark mode support
- Auto-save to localStorage
- YAML front matter support for document metadata (title, date, headers, footers)
- Click-to-navigate between editor and preview panes
- Clipboard operations (copy/paste)
- Markdown syntax guide panel
- Table of Contents panel with auto-generation and navigation

## Target Users

Developers, technical writers, and content creators who need a fast, browser-based markdown editing experience with immediate visual feedback.

## Key Differentiators

- No backend required - runs entirely in the browser
- Accurate line mapping between source and rendered output
- Configurable PDF export with fine-grained font control
- Multiple preview style themes
- Automatic TOC generation with PDF export integration
